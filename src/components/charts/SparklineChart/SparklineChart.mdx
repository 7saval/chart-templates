# SparklineChart

작은 영역에서 값의 흐름을 간단하게 보여주는 미니 차트 컴포넌트입니다. 숫자 배열(`data`)을 입력받아 추세를 시각적으로 표현하며, `height`로 표시 영역의 높이를 조절하고 `status` 값에 따라 정상/경고/위험 상태를 색상으로 구분하여 나타낼 수 있습니다. 카드나 테이블 등 좁은 공간에 삽입되어 값의 변화 추이를 빠르게 파악할 수 있도록 돕는 용도로 사용됩니다.

## Props

| Name     | Type          | Required | Default |
| -------- | ------------- | -------- | ------- |
| `data`   | `number[]`    | yes      | -       |
| `height` | `number`      | no       | 32      |
| `status` | `StatusLevel` | no       | info    |
